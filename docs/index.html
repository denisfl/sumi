Zero dependencies

Single static binary. No libraries, no runtime, no sudo required.

Multi-platform

macOS (Apple Silicon & Intel), Linux x86_64, Raspberry Pi arm64/armv7.

Tokyo Night TUI

ANSI TrueColor, bento-grid layout, Unicode box-drawing borders.

Raspberry Pi

ARM/GPU frequency, throttle status via vcgencmd.

JSON output

Pipe-friendly machine-readable snapshot. Works with jq.

Watch mode

Continuous refresh at configurable interval. --watch --interval 3

Install

macOS — Apple Silicon
curl -L https://github.com/denisfl/sumi/releases/latest/download/sumi-darwin-arm64.tar.gz | tar xz sudo mv sumi /usr/local/bin/
Linux — x86_64
curl -L https://github.com/denisfl/sumi/releases/latest/download/sumi-linux-amd64.tar.gz | tar xz sudo mv sumi /usr/local/bin/
Raspberry Pi — arm64
curl -L https://github.com/denisfl/sumi/releases/latest/download/sumi-linux-arm64.tar.gz | tar xz sudo mv sumi /usr/local/bin/
Build from source (Go 1.22+)
git clone https://github.com/denisfl/sumi.git cd sumi go build -o sumi ./cmd/monitor
